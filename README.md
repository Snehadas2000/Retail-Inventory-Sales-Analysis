# Retail-Inventory-Sales-Analysis
End-to End retail Analysis and sales using Excel, PowerBi, Python and SQL.

# Overview
This project analyzes retail inventory and sales performance to evaluate stock allocation efficiency and sales performance across stores and regions.

# Tools Used
Excel  
SQL  
Python (Pandas, NumPy, Matplotlib)  
Power BI  

# Key Analysis

• Inventory efficiency analysis  
• Sell-through rate calculation  
• Stock vs Sales comparison  
• Regional sales performance  
• Store-level performance evaluation  

# Dashboard Pages

Executive Summary  
Attribute Performance Analysis  
Regional Analysis  
Store Deep Dive  

# Key Insights

• Identified overstocked and underperforming product categories  
• Analyzed regional sales contribution  
• Evaluated store-level inventory efficiency  
• Provided recommendations for inventory rebalancing
